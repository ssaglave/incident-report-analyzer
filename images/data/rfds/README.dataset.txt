This dataset was created to be used for a YOLO-based Fire Detection Model, particularly in an indoor setting.

Images are in 416x416 format without any data augmentation/transformation.

Notes:
 - Class '0' = 'fire'
    Some images weren't annotated properly but the bounding boxes box fires.
